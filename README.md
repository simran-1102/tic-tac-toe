# tic-tac-toe

A responsive Tic-Tac-Toe game built with HTML, CSS, and JavaScript featuring both Player vs Player and Player vs AI modes.

1.Features  Two Game Modes:

 Player vs Player (PvP)

Player vs AI (with smart move logic)

2.Interactive UI:

 Clean, modern design

 Visual feedback for moves

 Winning combination highlight

3.Game Mechanics:

Turn-based gameplay

Win/draw detection

Game status updates

4.Responsive Design:

 Works on mobile and desktop

 Adapts to different screen sizes

Technologies Used  HTML5 - Game structure

CSS3 - Styling and animations

JavaScript - Game logic and interactivity

How to Play  Choose your game mode (PvP or vs AI)

Click any empty cell to place your mark (X goes first)

Try to get 3 of your marks in a row (horizontally, vertically, or diagonally)

Click "Reset Game" to start over
